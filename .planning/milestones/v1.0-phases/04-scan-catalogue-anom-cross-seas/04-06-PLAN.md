    `kernel.rs`:
    - `#[inline] pub(super) fn arch_lm_test(returns: &[f64], lag: usize) -> ArchLmResult` — implements Engle (1982):
      1. residuals_squared = (returns - mean(returns)).² (mean of returns is the regression intercept under homoskedasticity null)
      2. Construct the AR(lag) regression on squared residuals: u²_t = α + Σ β_i u²_{t-i} + e_t, for t = lag+1..n
      3. Fit via nalgebra SMatrix OLS — design matrix is (n-lag) × (lag+1)
      4. R² of the regression
      5. LM stat = (n - lag) * R²
      6. LM p-value = 1 - ChiSquared(lag).cdf(LM)
      7. F-stat = R²/(1-R²) * (n - 2*lag - 1) / lag
      8. F p-value = 1 - FisherSnedecor(lag, n - 2*lag - 1).cdf(F)
    - `pub struct ArchLmResult { pub lm: f64, pub lm_pvalue: f64, pub f_stat: f64, pub f_pvalue: f64, pub lag: usize }`
    - statrs::distribution::ChiSquared + statrs::distribution::FisherSnedecor imports
    - debug_assert! invariants; sequential summation order
    - Hand-derived unit test: for a small input where R² is exactly computable (e.g., n=10, lag=2), verify LM = (10-2) * R² within 1e-12
