    `kernel.rs`:
    - `#[inline] pub(super) fn variance_ratio(returns: &[f64], k: usize, robust: bool) -> VrResult` — implements Lo-MacKinlay:
      1. mu = mean(returns)
      2. sigma_1² = Σ (r_t - mu)² / (n - 1)
      3. sigma_k² (overlapping) = (1 / (n*k - k + 1)) * Σ_{t=k}^{n} (Σ_{j=0..k-1} r_{t-j} - k*mu)² / k   — the overlapping unbiased variance per Lo-MacKinlay's m_k correction
      4. VR(k) = sigma_k² / sigma_1²
      5. Asymptotic variance of VR-1: under iid null, asy_var = (2*(2*k - 1)*(k - 1)) / (3*k*n)
      6. Heteroskedasticity-robust variance: per Lo-MacKinlay (1988) eq (13b), use Σ over j of weight_j * δ_hat_j where δ_hat_j = Σ (r_t - mu)²(r_{t-j} - mu)² / (Σ(r_t-mu)²)²
      7. z_stat = (VR(k) - 1) / sqrt(robust_or_asy_var)
      8. p_value = 2 * (1 - Φ(|z_stat|)) two-sided via statrs::distribution::Normal
