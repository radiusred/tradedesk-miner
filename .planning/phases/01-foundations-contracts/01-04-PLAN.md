<objective>
Implement the StdoutSink (the ONE writer to stdout), the stderr_emit module (the ONE writer of structured pre-flight errors to stderr), and the workspace `clippy.toml` that enforces `std::println!` / `std::eprintln!` / `std::print!` / `std::eprint!` / `std::dbg!` are never used anywhere except (if needed) inside these two modules.

**Purpose:** FOUND-02 (stdout = findings, stderr = logs, CI-enforced via clippy lint), OUT-01 (NDJSON on stdout, one envelope per line, flushed). The stdout discipline is the second-most-non-retrofittable contract in Phase 1 after the envelope schema — once scans (Phase 4+) and wrappers (Phase 6) ship, fixing accidental stdout pollution downstream is exponentially harder. Threat T-01-03 (stdout pollution leaking sensitive paths into the JSONL stream) is mitigated by establishing the single-writer pattern and the clippy gate together.

**Output:** Two writer modules and one clippy config. `cargo clippy --workspace --all-targets -- -D warnings` runs clean. The pattern is enforced workspace-wide from this plan forward.
</objective>

    //! Pre-flight error emitter — writes structured-error JSON to stderr.
    //!
    //! D-06 requires that pre-flight rejection (unknown scan, invalid parameter,
    //! missing required config, etc.) emit a single structured-error JSON line
    //! to stderr while leaving stdout empty. This module is the ONLY writer of
    //! such records; logging via `tracing` is a separate concern.

    use std::io::{self, Write};
    use crate::error::WireError;

    /// Write one WireError as a single JSON line to the provided writer.
    /// The caller chooses the writer — production code uses `io::stderr()`;
    /// tests use an in-memory `Vec<u8>`.
    pub fn write_preflight_error<W: Write>(out: &mut W, err: &WireError) -> io::Result<()> {
        serde_json::to_writer(&mut *out, err)
            .map_err(|e| io::Error::new(io::ErrorKind::Other, e))?;
        out.write_all(b"\n")?;
        out.flush()?;
        Ok(())
    }

    /// Production convenience: write to std::io::stderr().
    pub fn emit_to_stderr(err: &WireError) -> io::Result<()> {
        write_preflight_error(&mut io::stderr(), err)
    }

        use std::collections::BTreeMap;
